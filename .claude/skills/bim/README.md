# BIM skills

Thessar skills eru BIM-sertaekar fyrir BIMClaw-agent:

- `procore-tools` - Procore verkefni, RFIs, issues, comments
- `acc-tools` - ACC verkefni og issues
- `deadline-checker` - Sjalfvirk frestaeftirlit og vidvaranir
- `weekly-summary` - Vikuleg verkefnastada i samantekt

Skills eru geymdar undir `.claude/skills/bim/` til ad halda BIM breytingum adskildum fra kjarna NanoClaw.
